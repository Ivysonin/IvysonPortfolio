# Portfólio de Ivyson Thauan

**Transformando ideias em soluções escaláveis com Python e Flask**

## Sobre

Este projeto é o portfólio online de **Ivyson Thauan**, desenvolvedor Back-end com mais de **1 ano de experiência prática** em Python, Flask e desenvolvimento de APIs REST. O portfólio apresenta experiências profissionais, projetos, habilidades técnicas e interpessoais, além de contato direto para oportunidades.

O site foi construído com **HTML, CSS, TailwindCSS e JavaScript**, priorizando uma interface moderna, responsiva e interativa.

## Funcionalidades

* **Seções completas:** Início, Sobre, Habilidades, Experiência, Projetos, Formação e Contato.
* **Navegação fluida:** Menu fixo, menu mobile e rolagem suave.
* **Projetos em destaque:** Cada projeto com imagem, descrição, tecnologias e status.
* **Download de CV:** Botão direto para baixar o currículo em PDF.
* **Links diretos:** GitHub, LinkedIn e email.
* **Design dinâmico:** Animações sutis, shapes flutuantes e efeitos de hover.

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* TailwindCSS

### Ferramentas e Conceitos

* Design responsivo
* Boa experiência de usuário (UX)
* Metodologias ágeis para projetos

## Como usar

1. Clone o repositório:

```bash
git clone https://github.com/Ivysonin/IvysonPortfolio.git
```

2. Abra o arquivo `index.html` no navegador ou hospede em um servidor web.
